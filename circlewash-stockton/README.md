# Circle Wash React App

React + JavaScript + MUI implementation based on the Laundromat Figma file.

## Run

```bash
npm install
npm run dev
```

## Images

Place the project images in `public/images`. The app looks for these names first:

- `circle-wash-logo.png`, `logo.png`, or `circlewash-logo.png`
- `hero-neon.jpg`, `coming-winter.jpg`, or `laundromat-hero.jpg`
- `wash-and-fold.jpg`, `self-service.jpg`, `commercial-towels.jpg`
- `free-drying.jpg`, `drop-off.jpg`, `wash.jpg`, `pickup.jpg`
- `map.png`

`public/images/figma-reference.png` is included as a fallback reference image.
